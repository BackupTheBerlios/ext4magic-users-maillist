This list is for discussion about anything related to ext4magic.

ext4magic is a utility to undelete files from an ext3 or ext4 partition.

It is based on an idea of the tools ext3grep and extundelete, but the whole 
functionality was again developed from scratch, and ext4magic goes one step 
further.  It handles more file types, hardlinks and softlinks, has more 
possibilities of working with the Journal and filesystem, restore owner and 
group of files and directories, restore the modification time, can find moved 
and overwritten directories and files, can find more then one version of a 
file, search older directory versions, can get a lot of helpful information 
over the Journal and the contained data, and any more.

Please read the notes on the README files within the download package and use 
the manpage to better understand how to use this utility. Currently there are 
only a detailed documentation in German. 
http://openfacts2.berlios.de/wikide/index.php/BerliosProject:Ext4magic

If you have any comments or questions about ext4magic, or would like to assist 
in the development of this utility in any way, including by reporting a bug , 
helping write the code, or helping translation the doku, please post to this 
mailing list.
